# 棋譜メーカー（PWA）

将棋の棋譜スクリーンショットや KIF ファイルを、印刷・保存できる **A4 の棋譜用紙**に清書するアプリです。ホーム画面に追加すればアプリのように使えます。

## 中身

| ファイル | 役割 |
|---|---|
| `index.html` | アプリ本体（これ一つで全機能が動く） |
| `manifest.webmanifest` | アプリ名・アイコン・表示方法の定義 |
| `sw.js` | Service Worker（オフライン起動用） |
| `icon-192.png` / `icon-512.png` | ホーム画面アイコン |
| `icon-maskable.png` | Android の丸型・角丸アイコン用 |
| `apple-touch-icon.png` | iPhone のホーム画面アイコン |

アイコンは息子さん直筆の「将棋大好きエイリアン」です。

## GitHub Pages で公開する手順

駒花火（komahanabi-app）と同じやり方です。

1. GitHub で新しいリポジトリを作る（例：`kifu-maker`）
2. この zip の中身を **すべて**アップロード（フォルダごとではなく、ファイルを直下に置く）
3. リポジトリの **Settings → Pages** を開く
4. **Source** を「Deploy from a branch」、**Branch** を `main` / `(root)` にして Save
5. 数分待つと `https://<ユーザー名>.github.io/kifu-maker/` で公開される

### iPhone でホーム画面に追加

1. Safari で公開URLを開く
2. 共有ボタン（□に↑）→「**ホーム画面に追加**」
3. エイリアンのアイコンで追加される

## 動作メモ

- **スクショの読み取り**（AI）は**オンライン時のみ**動きます
- **KIF の読み込み**と**A4棋譜の作成・保存**は**オフラインでも**動きます
- 保存は「棋譜を長押し →『"ファイル"に保存』→ 好きなフォルダ」です

## 更新方法

アプリを直したときは、`sw.js` の `const CACHE = "kifu-maker-v1"` の番号を
`v2`, `v3`… と上げてからアップロードすると、古いキャッシュが更新されます。
